Library Management System

A complete library management solution for Odoo 19, designed to handle book catalogs, member management, and rental tracking with automatic overdue detection.

Key Features

  • Book catalog management with author relationships
  • Member registration and tracking
  • Rental management with checkout and return workflows
  • Automatic overdue detection via scheduled actions
  • Email reminders for overdue books
  • Comprehensive search and filtering options
  • Full audit trail with Odoo chatter integration

Models

  • Books: Track your library's book collection
  • Authors: Manage author information and their works
  • Members: Register and manage library members
  • Rentals: Handle book checkouts and returns

Technical Specifications

  • Target: Odoo 19
  • Dependencies: base, mail
